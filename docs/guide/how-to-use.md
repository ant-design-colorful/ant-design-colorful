## 📍 如何使用？

### 安装

```bash
npm i antd-colorful
## or
yarn add antd-colorful
```

### 使用

```js
import 'antd-colorful/index.css';
```

### 说明

- 如果想在自己项目中引用，可查看 Demo 写法，建议单独查看代码引入单独样式。
- 由于 特定属性 / 类型引起 `css` 变动而出现样式未生效或异常，建议采用 Demo 写法，或自行参照修正。
- 正逐步构想并添加，如果你有好的想法和建议，欢迎 PR ~
